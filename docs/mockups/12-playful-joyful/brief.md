# Mockup 12 — Playful Joyful

## Design Direction Commitment Protocol

1. **Who is the user emotionally?** A junior engineer who just got paged — nervous but the tool should make them feel supported
2. **What should using this feel like?** Friendly and approachable despite the seriousness — you want to keep clicking
3. **What design world?** Notion meets Figma meets a friendly ops tool — Lottie, Vercel's playful moments
4. **One adjective:** Delightful
5. **Strictly forbidden:** Seriousness, grey monoculture, anything corporate or intimidating

## Lens: Playful / Joyful

- **Typography:** Nunito (display, rounded), DM Sans (body)
- **Motion:** Bouncy entrances, springy hover states, cheerful micro-interactions
- **Palette:** Soft purple #8B5CF6, warm coral #FB923C, sky blue #38BDF8, mint #34D399, cream #FFFBEB
- **Layout:** Rounded card grid with colored accent strips, emoji-scale status icons, pill-shaped elements everywhere, stacked cards with offset shadows

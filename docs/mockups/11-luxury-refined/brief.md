# Mockup 11 — Luxury Refined

## Design Direction Commitment Protocol

1. **Who is the user emotionally?** A senior VP reviewing an incident summary — wants clarity, not noise
2. **What should using this feel like?** Nothing is accidental. Every element earned its place.
3. **What design world?** A high-end annual report or Apple product page — Stripe, Linear, Notion
4. **One adjective:** Deliberate
5. **Strictly forbidden:** Clutter, bright colors, anything generic or busy

## Lens: Luxury / Refined

- **Typography:** Instrument Serif (display), Inter (body, light weight)
- **Motion:** Slow fades, subtle parallax, elegant hover states
- **Palette:** Warm off-white #FAF9F6, charcoal #1C1917, muted gold #B8860B, stone #78716C
- **Layout:** Vertical scroll with generous whitespace, data presented as refined cards with thin hairline borders, asymmetric text blocks

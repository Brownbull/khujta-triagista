# Mockup 13 — Combined Final

## Design Direction
Combines the best of 3 mockups:
- **07 Ops Dashboard** — sidebar + compact panels + KPI strip + status bar (base layout)
- **08 AI Conversational** — chat timeline view for incident-as-conversation
- **12 Playful Joyful** — bouncy hover effects (spring cubic-bezier, scale, float)

## Views (switchable via sidebar)
1. **Incidents List** — table of all incidents, clickable rows
2. **Incident Detail** — KPI strip + panels (ops dashboard style)
3. **Incident Chat** — same incident shown as AI conversation timeline

## Anti-convergence
This is a functional prototype, not a style exercise. It proves the 3-view navigation concept.

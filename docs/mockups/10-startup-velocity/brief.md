# Mockup 10 — Startup Velocity

## Design Direction Commitment Protocol

1. **Who is the user emotionally?** An on-call engineer in crisis mode — adrenaline, urgency, needing fast answers
2. **What should using this feel like?** Things are moving fast and the tool is keeping up with you
3. **What design world?** A mission control center at a fast-moving startup — Vercel, Linear, Raycast
4. **One adjective:** Kinetic
5. **Strictly forbidden:** Muted colors, static layouts, anything that says "slow" or "finished"

## Lens: Startup Velocity

- **Typography:** Space Grotesk (display, bold), Inter (body)
- **Motion:** Fast directional slides, elements snap into place, gradient shimmer on key data
- **Palette:** Deep navy #0A0E27, electric violet #7C3AED, hot coral #FF6B6B, lime #84CC16
- **Layout:** Full-width hero strip with live-feeling KPIs, horizontal scrolling action cards, floating resolve bar

# Project Values

- **V1 — No Ghost States:** Every async operation (pipeline stages, notifications, ticket updates) has an explicit, visible status. No fire-and-forget without tracking. `session`
- **V2 — Demo or Die:** If it won't show in the 3-minute demo, deprioritize it. Ship the demo path first. `session`
- **V3 — Deliver, Don't Design:** When caught between "make it elegant" and "make it work", choose work. Refactor only if it unblocks the next phase. `story`
